## Usage

```
# create txt file
$ cat << EOF > simple.txt
    +--------+   +-------+    +-------+
    |        | --+ ditaa +--> |       |
    |  Text  |   +-------+    |diagram|
    |Document|   |!magic!|    |       |
    |     {d}|   |       |    |       |
    +---+----+   +-------+    +-------+
        :                         ^
        |       Lots of work      |
        +-------------------------+
EOF

# generate png file
$ docker run --rm -it  -v `pwd`:/root chai2010/ditaa simple.txt output.png 
```
